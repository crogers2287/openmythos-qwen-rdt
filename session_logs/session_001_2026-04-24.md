## Implementation Attempts (v1-v11)

|---------|----------|--------|
| v1-v7 | RecurrentDepthWrapper calling layers individually | Hung (per-layer gradient checkpointing explosion) |

| SFT | Clean Unsloth SFTTrainer, no RDT loop | **WORKED** — but ~50h ETA, no RDT architecture |

## Key Lesson
**You cannot modify the layer execution order of a model that Unsloth has heavily patched without breaking its gradient checkpointing, compiled kernels, and accelerate hooks.** Every approach that called individual layers outside the model's own forward path either hung or deadlocked.

## 24-Hour Config Solution
**R=8, seq_len=1024, grad_accum=1 → estimated 21.3 hours**

## Corrected Compute Model
```
RDT effective depth = Prelude(4) + Core(4) × R + Coda(4) = 8 + 4R
R=8  → 40 executions → 1.00x baseline
R=16 → 72 executions → 1.80x baseline (too expensive for 24h)
```
